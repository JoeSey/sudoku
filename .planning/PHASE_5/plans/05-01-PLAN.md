# Plan 05-01: Zen Mode & UI Polish

## Goal
Implement the "Zen Mode" toggle and its visual effects, including hiding the timer, mistakes, and keypad progress, while disabling assists like conflict highlights.

## Tasks
1. **Extend Game Store**: 
   - Add `isZenMode: boolean` to `GameState`.
   - Add `toggleZenMode: () => void` action to `useGameStore`.
2. **Zen Toggle Button**:
   - Add a minimalist "Zen" button/toggle to the main UI (likely near the timer or settings).
3. **UI Conditional Rendering**:
   - Update `GameInfo` to hide the Timer and Mistake counter when `isZenMode` is true.
   - Update `Keypad` to hide the digit progress pie-charts when `isZenMode` is true.
4. **Assist Disabling**:
   - Update `SudokuGrid` and `SudokuCell` to suppress row/col/block highlights and conflict (red) highlights when `isZenMode` is true.
5. **Verification**:
   - Toggle Zen Mode and verify all specified UI elements disappear.
   - Confirm that placing a conflicting number does NOT show a red highlight in Zen Mode.

## Technical Details
- **Store**: `isZenMode` should be persisted in `sudoku-storage`.
- **Styling**: Ensure the Zen toggle itself is visually subtle to fit the theme.
