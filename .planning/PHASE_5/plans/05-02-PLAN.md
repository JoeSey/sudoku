# Plan 05-02: Logic Engine Integration & Assisted Tracking

## Goal
Integrate the `sudoku-core` library for logic analysis and implement the "Assisted" tracking for games where hints are used.

## Tasks
1. **Dependency Installation**:
   - Install `sudoku-core`.
2. **Logic Utility Service**:
   - Create `src/utils/hintService.ts` to wrap `sudoku-core.hint()`.
   - Implement mapping from `sudoku-core` strategies to human-friendly titles and cells.
3. **State Extension**:
   - Add `isAssisted: boolean` to `GameState`.
   - Update `bestTimes` recording to include the `isAssisted` flag.
4. **Assisted UI Indication**:
   - Show the "A" label in the "New Game" and "Best Times" screens for assisted games.
5. **Hint Logic Prototype**:
   - Implement a basic "Naked Single" detection in the hint service.
6. **Verification**:
   - Solve a puzzle with a hint and verify the "Assisted" label is recorded and displayed correctly in the best times table.

## Technical Details
- **sudoku-core**: Ensure the 9x9 grid representation matches the library's expectations (likely a 1D array of 81 numbers/nulls).
- **Persistence**: Ensure `isAssisted` is reset on `initGame`.
